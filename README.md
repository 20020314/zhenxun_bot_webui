# 开发环境使用方式（请先进入webui根目录文件夹下再使用命令）

****

### 开发环境需要下载nodejs，未安装请移步链接 https://nodejs.org/zh-cn/download/



## 使用npm（nodejs自带）

### 下载webui开发依赖

```
npm install
```

### 启动开发服务器
```
npm run serve
```

### 打包生产环境资源
```
npm run build
```

## 使用yarn（需要下载）

### 使用npm安装yarn

```
npm install --global yarn
```

### 下载webui开发依赖

```
yarn install
```

### 启动开发服务器
```
yarn serve
```

### 打包生产环境资源
```
yarn build
```
